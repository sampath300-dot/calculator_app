# Calculator App

A modern calculator app built with Flutter.

## Features

* Addition, subtraction, multiplication and division
* Custom wallpaper/background support
* Dark theme UI
* Backspace button
* Android APK support
* Built using Flutter and Dart

## Installation

```bash
flutter pub get
flutter run
```

## Build APK

```bash
flutter build apk
```

## Technologies Used

* Flutter
* Dart
* image_picker
* math_expressions
